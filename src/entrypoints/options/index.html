Icon Labeler — Advanced

Writes no labels at page load. Instead, a single aria-label is injected onto an icon button/link only while it has keyboard focus, then removed on blur — so the page DOM stays untouched during invisible bot-challenge checks (Cloudflare, DataDome, Vercel BotID). Uncheck for the legacy "label everything at load" mode: it also labels non-focusable icons, but its DOM writes are visible to site integrity monitors and can trip checkpoints.
